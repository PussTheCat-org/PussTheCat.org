PussTheCat.org

PussTheCat.org provides access to open source services that are focused on privacy and video games servers.

PussTheCat.org is based entirely on a non-profit basis, however since it's not the most sustainable thing, we strongly encourage you to donate if you can.

Public Matrix discussion and status room: #pussthecat.org:pussthecat.org

Status page: status.pussthecat.org

Services:



Invidious

Invidious is a way to use YouTube without being tracked.



Nitter

Nitter is a way to use Twitter without being tracked.



SearX

Searx is a way to use any search engine without being tracked.



Teddit

Teddit is a way to use Reddit without being tracked.





Libreddit

Libreddit is another way to use Reddit without being tracked.



Lingva

Lingva is a way to use Google Translate without being tracked.



SimplyTranslate

SimplyTranslate is a way to use Google Translate, DeepL and ICIBA Translate without being tracked.



rimgo

rimgo is a way to use Imgur without being tracked.





Librarian

Librarian is a way to use Odysee (LBRY) without being tracked.



Scribe

Scribe is a way to use Medium without being tracked.



ProxiTok

ProxiTok is a way to use TikTok without being tracked.



PeerTube

PeerTube is a self-hosted ActivityPub-federated video streaming platform using P2P directly in your web browser.

Note: This instance is focused on re-uploading deleted YouTube videos.





Libremdb

Librembd is a way to use IMDB without being tracked.



Quetre

Quetre is a way to use Quora without being tracked.



BreezeWiki

BreezeWiki is a way to use Fandom without being tracked.

Call of Duty 4 - Kill the King

Kill the King (KtK) is a mod which changes the COD4 experience dramatically by dividing teams in two groups - assassins and guardians. At first, one lone assassin must try to kill the king - whilst avoiding the guardians. Killing a guardian will respawn him as an assassin, who can then help to kill the King.
