Minecraft

PussTheCat.org provides access to multiple private or semi-private servers.

Features


  • All servers are running on a powerful server, absolutely no lags ever happen
  • All servers have a daily (sometimes more often) backup task running so that things don't go wrong
  • All servers are available through a whitelist so that the players have a good experience
  • All servers requires a premium Minecraft accounts, "cracks" are disallowed, so that the players have a better experience, and are able play on multiples PCs with the same "character"
  • All servers have a "low" max-world-size by default (between 1500 and 2000), so that the world download isn't massive, its value is increased on demand, or in case new world generation is needed
  • All servers have their world spawn set to 0 - 0 for consistency purpose
  • All servers have their difficulty set to normal
